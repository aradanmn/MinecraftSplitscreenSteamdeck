**Title:** Would you consider adding a license (e.g. MIT)?

Hi — thanks for building this, your splitscreen installer is what got the whole thing
working for me on the Steam Deck. I've been maintaining a fork
([aradanmn/MinecraftSplitscreenSteamdeck](https://github.com/aradanmn/MinecraftSplitscreenSteamdeck))

good chunk of the installer and launcher plumbing still traces back to your original code,
with credit to you in the README.

Right now the repo has no license, which technically means it's all-rights-reserved and
others can't reuse it beyond viewing/forking on GitHub. Would you be open to adding a
permissive license — MIT would be my suggestion — so the original code is clear for anyone

happy to open a PR with an MIT file if that's easier.

Totally understand if you'd rather not — either way, thanks for the tool and for putting it
out there.
